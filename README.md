# Comparing Rule-Based, Supervised, and Unsupervised Approaches for Anomaly Detection in System Logs

## Overview

This repository contains the code, datasets, experiments, and research artifacts for my CS 6625 Specialized Study at Troy University.

The study evaluates the effectiveness of traditional rule-based monitoring compared to supervised and unsupervised machine learning approaches for anomaly detection in distributed system logs.

## Dataset

The primary dataset used in this study is the HDFS (Hadoop Distributed File System) log dataset.

HDFS is a distributed file storage system designed to run across clusters of machines. The dataset contains system log events generated by HDFS operations and includes labeled anomalous behavior, making it well suited for evaluating anomaly detection techniques.

## Research Objective

The goal of this research is to determine whether machine learning approaches can improve anomaly detection performance when compared to traditional threshold-based monitoring methods.

The following approaches will be evaluated:

- Rule-Based Detection
- Logistic Regression
- Random Forest
- Isolation Forest

## Evaluation Metrics

Performance will be compared using:

- Accuracy
- Precision
- Recall
- F1 Score
